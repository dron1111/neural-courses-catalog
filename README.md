# Каталог курсов по нейросетям

Каталог курсов с фильтрами, поиском и партнерскими ссылками.

## Быстрый старт

1. Установите зависимости:
```bash
pip install -r requirements.txt

API
GET /api/courses - список курсов с фильтрами

GET /api/course/{slug} - информация о курсе

GET /api/admin/courses?token={token} - список всех курсов (админка)

Деплой
Проект настроен для деплоя на Render.com. Просто подключите репозиторий.
